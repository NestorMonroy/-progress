# knoReact ListView Component for React


The [React ListView](listview), part of knoReact, enables you to display a custom layout of data items.

## React ListView

Among the features which the knoReact ListView component delivers are:

-   [Paging](components/listview/paging/?utm_medium=referral&utm_source=npm&utm_campaign=kno-ui-react-trial-npm-listview)&mdash;The knoReact ListView can be easily integrated with the [knoReact Pager](components/datatools/pager/?utm_medium=referral&utm_source=npm&utm_campaign=kno-ui-react-trial-npm-listview) to split the content into multiple pages.
-   [Endless scrolling](components/listview/endless-scrolling/?utm_medium=referral&utm_source=npm&utm_campaign=kno-ui-react-trial-npm-listview)&mdash;The knoReact ListView can achieve `endless scrolling` allowing to dynamically display large collection of items.
-   [Filtering](components/listview/filtering/?utm_medium=referral&utm_source=npm&utm_campaign=kno-ui-react-trial-npm-listview)&mdash;The knoReact ListView can be easily integrated with the [knoReact Filter](components/datatools/filter/?utm_medium=referral&utm_source=npm&utm_campaign=kno-ui-react-trial-npm-listview) to filter the data in the list.
-   [Editing](components/listview/editing/?utm_medium=referral&utm_source=npm&utm_campaign=kno-ui-react-trial-npm-listview)&mdash;TThe knoReact ListView allow editing the items on the list.
-   [Theme support](components/styling/?utm_medium=referral&utm_source=npm&utm_campaign=kno-ui-react-trial-npm-listview)&mdash;The knoReact ListView, as well as all 100+ components in the knoReact suite, are styled in three polished themes (Bootstrap, Material, and Default) and can be further customized to match your specific design guidelines.
