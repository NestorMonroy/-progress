# knoReact Gantt Component for React



The [React Gantt](gantt-chart), part of knoReact, enables the display of self-referencing tabular data and allows sorting, filtering, and data editing when required.

## React Gantt

Among the features which the knoReact Gantt component delivers are:

-   [Filtering](components/gantt/filtering/?utm_medium=referral&utm_source=npm&utm_campaign=kno-ui-react-trial-npm-gantt)&mdash;The knoReact Gantt enables you to display only those Gantt records which meet specified criteria.
-   [Sorting](components/gantt/sorting/?utm_medium=referral&utm_source=npm&utm_campaign=kno-ui-react-trial-npm-gantt)&mdash;The knoReact Gantt enables you to sort single and multiple data-bound columns.
-   [Transformation of flat data](components/gantt/flat-data/?utm_medium=referral&utm_source=npm&utm_campaign=kno-ui-react-trial-npm-gantt)&mdash;Provides a built-in function to convert flat data to a tree that can be visualized by the component.
-   [Globalization](components/gantt/globalization/?utm_medium=referral&utm_source=npm&utm_campaign=kno-ui-react-trial-npm-gantt)&mdash;By using the available globalization options in knoReact, you can translate the Gantt messages by adapting them to specific culture locales.
-   [Theme support](components/styling/?utm_medium=referral&utm_source=npm&utm_campaign=kno-ui-react-trial-npm-gantt)&mdash;The knoReact Gantt, as well as all 100+ components in the knoReact suite, are styled in three polished themes (Bootstrap, Material, and Default) and can be further customized to match your specific design guidelines.
