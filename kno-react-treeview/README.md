# knoReact TreeView Component for React

The [React TreeView](treeview), part of knoReact, displays hierarchical data in a traditional tree structure, supports user interaction through mouse or touch events, and performs reordering operations by using the drag-and-drop functionality.

## React TreeView

Among the features which the knoReact TreeView component delivers are:

-   [Data binding](components/treeview/data-binding/?utm_medium=referral&utm_source=npm&utm_campaign=kno-ui-react-trial-npm-treeview)&mdash;Works with various types of data.
-   [Persisting the focus](components/treeview/data-reload/?utm_medium=referral&utm_source=npm&utm_campaign=kno-ui-react-trial-npm-treeview)&mdash;The TreeView enables you to keep the focus on the currently focused item while reloading the data.
-   [Expanding](components/treeview/expansion/?utm_medium=referral&utm_source=npm&utm_campaign=kno-ui-react-trial-npm-treeview)&mdash;The Allows expanding and collapsing items in the data tree.
-   [Selection](components/treeview/selection/?utm_medium=referral&utm_source=npm&utm_campaign=kno-ui-react-trial-npm-treeview)&mdash;The Allows selecting items in the data tree.
-   [Checkboxes](components/treeview/checkboxes/?utm_medium=referral&utm_source=npm&utm_campaign=kno-ui-react-trial-npm-treeview)&mdash;Allows the user to select nodes by using node checkboxes.
-   [Dragging and dropping](components/treeview/drag-drop/?utm_medium=referral&utm_source=npm&utm_campaign=kno-ui-react-trial-npm-treeview)&mdash;The drag-and-drop functionality enables the user to move the TreeView items by dragging and dropping them within a single tree or across multiple trees.
-   [Custom rendering](components/treeview/custom-rendering/?utm_medium=referral&utm_source=npm&utm_campaign=kno-ui-react-trial-npm-treeview)&mdash;Allows customization of certain or all TreeView nodes.
-   [Keyboard navigation](components/treeview/keyboard-navigation/?utm_medium=referral&utm_source=npm&utm_campaign=kno-ui-react-trial-npm-treeview)&mdash;Built-in keyboard shortcuts for usability and navigation in the component.
-   [Accessibility](components/treeview/accessibility/?utm_medium=referral&utm_source=npm&utm_campaign=kno-ui-react-trial-npm-treeview)&mdash;The TreeView is accessible by screen readers and provides full WAI-ARIA support.
-   [Theme support](components/styling/?utm_medium=referral&utm_source=npm&utm_campaign=kno-ui-react-trial-npm-treeview)&mdash;The knoReact TreeView, as well as all 100+ components in the knoReact suite, are styled in three polished themes (Bootstrap, Material, and Default) and can be further customized to match your specific design guidelines.
