The [React Excel Export feature](excelexport), part of knoReact, allows you to export and save data to Excel files, to customize the output cells and columns, and to filter the data after the export.

## React Excel Export

Among the features which the knoReact Excel Export component delivers are:

-   [Columns](components/excelexport/columns/?utm_medium=referral&utm_source=npm&utm_campaign=kno-ui-react-trial-npm-excelexport)&mdash;The Excel Export component supports options for customization of the columns that will be exported to Excel.
-   [Cells](components/excelexport/cells/?utm_medium=referral&utm_source=npm&utm_campaign=kno-ui-react-trial-npm-excelexport)&mdash;The Excel Export component supports options for customizing the cells that will be exported to Excel.
-   [Workbook customization](components/excelexport/customization/?utm_medium=referral&utm_source=npm&utm_campaign=kno-ui-react-trial-npm-excelexport)&mdash;The Excel Export component provides full access to the workbook allowing customization everything from the sheets to the cell styles.
-   [Filtering](components/excelexport/filtering/?utm_medium=referral&utm_source=npm&utm_campaign=kno-ui-react-trial-npm-excelexport)&mdash;The Excel Export component enables you to filter the data in the columns of the exported Excel file.
-   [Saving files on the server](components/excelexport/saving/?utm_medium=referral&utm_source=npm&utm_campaign=kno-ui-react-trial-npm-excelexport)&mdash;The Excel Export component enables you to send the generated Excel file to a remote service as base64-encoded string.
-   [Theme support](components/styling/?utm_medium=referral&utm_source=npm&utm_campaign=kno-ui-react-trial-npm-excelexport)&mdash;The knoReact Excel Export, as well as all 100+ components in the knoReact suite, are styled in three polished themes (Bootstrap, Material, and Default) and can be further customized to match your specific design guidelines.
