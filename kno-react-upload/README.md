# knoReact Upload Component for React


## React Upload

Among the features which the knoReact Upload component delivers are:

-   [Disabled Upload](components/upload/disabled-state/?utm_medium=referral&utm_source=npm&utm_campaign=kno-ui-react-trial-npm-upload)&mdash;Completely disable the upload functionality in certain conditions.
-   [Modes of operation](components/upload/modes/?utm_medium=referral&utm_source=npm&utm_campaign=kno-ui-react-trial-npm-upload)&mdash;Control which file types can be uploaded or execute follow-up logic based on certain scenario.
-   [Credentials and additional data](components/upload/credentials/?utm_medium=referral&utm_source=npm&utm_campaign=kno-ui-react-trial-npm-upload)&mdash;Send cookies and/or headers for cross-site requests or additional data.
-   [File processing](components/upload/file-processing/?utm_medium=referral&utm_source=npm&utm_campaign=kno-ui-react-trial-npm-upload)&mdash;Provides different configuration options for controlling the way files are processed during upload. For example, automatic upload, single or multiple files upload, or upload of batches of files.
-   [File restrictions](components/upload/restrictions/?utm_medium=referral&utm_source=npm&utm_campaign=kno-ui-react-trial-npm-upload)&mdash;Restrict certain file uploads, based on their file size or extension.
-   [Dragging and dropping](components/upload/drag-and-drop/?utm_medium=referral&utm_source=npm&utm_campaign=kno-ui-react-trial-npm-upload)&mdash;For easy use and delightful experiences, the component has built-in drag and drop functionality.
-   [Custom renderer](components/upload/custom-rendering/?utm_medium=referral&utm_source=npm&utm_campaign=kno-ui-react-trial-npm-upload)&mdash;You can apply your own look and feel to the uploaded files list.
-   [External Drop Zone](components/upload/external-drop-zone/?utm_medium=referral&utm_source=npm&utm_campaign=kno-ui-react-trial-npm-upload)&mdash;The ExternalDropZone helps users to drag and drop files from their file systems to a dedicated user interface. The files are then sent to the specified server handlers, which are configured to receive them.
-   [Globalization](components/upload/globalization/?utm_medium=referral&utm_source=npm&utm_campaign=kno-ui-react-trial-npm-upload)&mdash;By using the available globalization options in knoReact, you can translate the Upload messages by adapting them to specific culture locales.
-   [Keyboard navigation](components/upload/keyboard-navigation/?utm_medium=referral&utm_source=npm&utm_campaign=kno-ui-react-trial-npm-upload)&mdash;Built-in keyboard shortcuts for usability and navigation in the component.
-   [Theme support](components/styling/?utm_medium=referral&utm_source=npm&utm_campaign=kno-ui-react-trial-npm-upload)&mdash;The knoReact Upload, as well as all 100+ components in the knoReact suite, are styled in three polished themes (Bootstrap, Material, and Default) and can be further customized to match your specific design guidelines.
