# knoReact WYSIWYG Editor Component for React

[The React Editor](editor), part of knoReact, enables users to create rich textual content through a What-You-See-Is-What-You-Get (WYSIWYG) interface and delivers a set of tools for creating, editing, and formatting of text, paragraphs, lists, and other HTML elements.

## React HTML Editor

Among the features which the knoReact Editor component delivers are:

-   [Ready-to-use, customizable toolset](components/editor/tools/?utm_medium=referral&utm_source=npm&utm_campaign=kno-ui-react-trial-npm-editor)&mdash;The Editor provides a rich set of built-in, user-interface tools. It also enables you to customize each one of them and/or add custom ones.
-   [Custom rendering](components/editor/custom-rendering/?utm_medium=referral&utm_source=npm&utm_campaign=kno-ui-react-trial-npm-editor)&mdash;The Editor provides options for customizing the way it renders its elements—for example, HTML wrapper element, tools, and dialogs.
-   [Get and/or set HTML content](components/editor/content/?utm_medium=referral&utm_source=npm&utm_campaign=kno-ui-react-trial-npm-editor)&mdash;You can programmatically get the HTML content of the Editor and/or set an updated version back.
-   [Sanitizing pasted HTML content](components/editor/paste/?utm_medium=referral&utm_source=npm&utm_campaign=kno-ui-react-trial-npm-editor)&mdash;A set of built-in functions look after sanitizing the content which the user pastes in the content area of the editor.
-   [Plugins](components/editor/plugins/?utm_medium=referral&utm_source=npm&utm_campaign=kno-ui-react-trial-npm-editor)&mdash;Various ways to extend and customize the functionality and features of the editor by using different plugins.
-   [Modifying the schema](components/editor/schema/?utm_medium=referral&utm_source=npm&utm_campaign=kno-ui-react-trial-npm-editor)&mdash;By modifying the default, built-in schema, you can add additional types of nodes and marks as well as edit and remove the existing ones. This results in more tailored and customizable experience for the users.
-   [Globalization](components/editor/globalization/?utm_medium=referral&utm_source=npm&utm_campaign=kno-ui-react-trial-npm-editor)&mdash;By using the available globalization options in knoReact, you can translate the Editor messages by adapting them to specific culture locales.
-   [Theme support](components/styling/?utm_medium=referral&utm_source=npm&utm_campaign=kno-ui-react-trial-npm-editor)&mdash;The knoReact Editor, as well as all 100+ components in the knoReact suite, are styled in three polished themes (Bootstrap, Material, and Default) and can be further customized to match your specific design guidelines.
