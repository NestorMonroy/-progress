# knoReact Form Component for React



React Form, part of knoReact, is a small and fast package for form state management with zero dependencies.

## React Form

Among the features which the knoReact Form component delivers are:

-   [Custom components](components/form/custom-components/?utm_medium=referral&utm_source=npm&utm_campaign=kno-ui-react-trial-npm-form)&mdash;You can render custom components in the [knoReact field](components/form/api/Field/?utm_medium=referral&utm_source=npm&utm_campaign=kno-ui-react-trial-npm-form) component by using the built-in properties of the [`FieldRenderProps`](components/form/api/FieldRenderProps/?utm_medium=referral&utm_source=npm&utm_campaign=kno-ui-react-trial-npm-form) interface and any custom properties.
-   [Theme support](components/styling/)&mdash;The knoReact Form, as well as all 100+ components in the knoReact suite, are styled in three polished themes (Bootstrap, Material, and Default) and can be further customized to match your specific design guidelines.
