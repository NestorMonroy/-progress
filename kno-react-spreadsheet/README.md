## React Spreadsheet

The [React Spreadsheet library](spreadsheet), part of knoReact, allows you to edit and visualize tabular data by using cell formatting options, styles, and themes.

Among the features which the knoReact Spreadsheet component delivers are:

-   Import and export of excel files
-   Set custom styles to cells, rows, columns or regions
-   Resize rows and columns
-   Toolbar and tools configuration
-   Built-in formulas
-   Keyboard navigation
