# knoReact TreeList Component for React

## React TreeList

Among the features which the knoReact TreeList component delivers are:

-   [Editing](components/treelist/editing/?utm_medium=referral&utm_source=npm&utm_campaign=kno-ui-react-trial-npm-treelist)&mdash;You can create, update, and delete the data records of the TreeList.
-   [Filtering](components/treelist/filtering/?utm_medium=referral&utm_source=npm&utm_campaign=kno-ui-react-trial-npm-treelist)&mdash;The knoReact TreeList enables you to display only those TreeList records which meet specified criteria.
-   [Sorting](components/treelist/sorting/?utm_medium=referral&utm_source=npm&utm_campaign=kno-ui-react-trial-npm-treelist)&mdash;The knoReact TreeList enables you to sort single and multiple data-bound columns.
-   [Responsive column widths](components/treelist/column-widths/?utm_medium=referral&utm_source=npm&utm_campaign=kno-ui-react-trial-npm-treelist)&mdash;The component behaves like an HTML table and expands to the width of the parent element automatically.
-   [Selection](components/treelist/selection/?utm_medium=referral&utm_source=npm&utm_campaign=kno-ui-react-trial-npm-treelist)&mdash;The TreeList enables the user to select a single row or multiple rows and provides checkbox and row-click selection options.
-   [Customizable cells](components/treelist/cells/?utm_medium=referral&utm_source=npm&utm_campaign=kno-ui-react-trial-npm-treelist)&mdash;Provides options for customizing the content by defining different types of cells.
-   [Transformation of flat data](components/treelist/flat-data/?utm_medium=referral&utm_source=npm&utm_campaign=kno-ui-react-trial-npm-treelist)&mdash;Provides a built-in function to convert flat data to a tree that can be visualized by the component.
-   [Globalization](components/treelist/globalization/?utm_medium=referral&utm_source=npm&utm_campaign=kno-ui-react-trial-npm-treelist)&mdash;By using the available globalization options in knoReact, you can translate the TreeList messages by adapting them to specific culture locales.
-   [Theme support](components/styling/?utm_medium=referral&utm_source=npm&utm_campaign=kno-ui-react-trial-npm-treelist)&mdash;The knoReact TreeList, as well as all 100+ components in the knoReact suite, are styled in three polished themes (Bootstrap, Material, and Default) and can be further customized to match your specific design guidelines.
