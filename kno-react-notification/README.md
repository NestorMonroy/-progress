# knoReact Notification Component for React


The [React Notification](notification), part of , renders a brief message to the user with information about the status of an application process and enables you to animate its rendering and control its position.

How to start:

```sh
npm i @progress/kno-react-notification
```

## React Notification

Among the features which the knoReact Notification component delivers are:

-   [Notification types](components/notification/types/?utm_medium=referral&utm_source=npm&utm_campaign=kno-ui-react-trial-npm-notification)&mdash;Depending on the outcome of application operations and processes, the Notification enables you to display different types of notifications, like `info`, `error`, `warning`, and more.
-   [Animations](components/notification/animations/?utm_medium=referral&utm_source=npm&utm_campaign=kno-ui-react-trial-npm-notification)&mdash;The Notification allows you to add opening and closing animations to it by using the knoReact Animation components from the [@progress/kno-react-animation package](https://www.npmjs.com/package/@progress/kno-react-animation).
-   [Positioning](components/notification/positioning/?utm_medium=referral&utm_source=npm&utm_campaign=kno-ui-react-trial-npm-notification)&mdash;You can configure the position of the Notification through a property.
-   [Theme support](components/styling/?utm_medium=referral&utm_source=npm&utm_campaign=kno-ui-react-trial-npm-notification)&mdash;The knoReact Notification, as well as all 120+ components in the knoReact suite, are styled in three polished themes (Bootstrap, Material, Fluent, and Default) and can be further customized to match your specific design guidelines.

How to use the Notification component in your apps:

```tsx
import { Notification, NotificationGroup } from "@progress/kno-react-notification";
...
<NotificationGroup>
    <Notification>
        /** Provide the contents of your notification.
        *
        */
    </Notification>
    /**
    * Add as many Notifications as needed.
    */
</NotificationGroup>
```
