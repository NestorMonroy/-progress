# knoReact Map Library for React


The [React Map](map), part of knoReact, displays tile and vector maps.

## React Map Component

[The React Map component](components/map/?utm_medium=referral&utm_source=npm&utm_campaign=kno-ui-react-trial-npm-map) found in knoReact is a powerful data visualization tool that can fit into any application. This one component includes 4 different types of map layer types. Available layer types include:

-   [Tile Layer](components/map/layers/tile/?utm_medium=referral&utm_source=npm&utm_campaign=kno-ui-react-trial-npm-map)
-   [Shape (GeoJSON) Layer](components/map/layers/shape/?utm_medium=referral&utm_source=npm&utm_campaign=kno-ui-react-trial-npm-map)
-   [Bubble Map Layer](components/map/layers/bubble/?utm_medium=referral&utm_source=npm&utm_campaign=kno-ui-react-trial-npm-map)
-   [Marker Layer](components/map/layers/marker/?utm_medium=referral&utm_source=npm&utm_campaign=kno-ui-react-trial-npm-map)
