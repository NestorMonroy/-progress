The [React OrgChart](orgchart), part of knoReact, renders a brief message to the user with information about the status of an application process and enables you to animate its rendering and control its position.

## React OrgChart

Among the features which the knoReact OrgChart component delivers are:

-   [OrgChart types](components/orgchart/types/?utm_medium=referral&utm_source=npm&utm_campaign=kno-ui-react-trial-npm-orgchart)&mdash;Depending on the outcome of application operations and processes, the OrgChart enables you to display different types of orgcharts, like `info`, `error`, `warning`, and more.

-   [Theme support](components/styling/?utm_medium=referral&utm_source=npm&utm_campaign=kno-ui-react-trial-npm-orgchart)&mdash;The knoReact OrgChart, as well as all 100+ components in the knoReact suite, are styled in three polished themes (Bootstrap, Material, and Default) and can be further customized to match your specific design guidelines.
