# knoReact Scheduler Component for React


The [React Scheduler](components/scheduler), part of knoReact, provides out-of-the-box support for timezone conversion, resources visualization, grouping, editing, and recurring events.

## React Scheduler

The knoReact Scheduler component is built on React from the ground up by developers with 19+ years of experience in making enterprise-ready components. This results in a React scheduling component that delivers lightning fast performance and is highly customizable.

Among the many features which the knoReact Scheduler delivers are:

-   [Multiple views](components/scheduler/views/?utm_medium=referral&utm_source=npm&utm_campaign=kno-ui-react-trial-npm-scheduler)&mdash;You can display the Scheduler events in any of the available Timeline, Day, Week, Month, or Agenda views.
-   [Editing](components/scheduler/editing/?utm_medium=referral&utm_source=npm&utm_campaign=kno-ui-react-trial-npm-scheduler)&mdash;You can create, update, and delete events displayed by the Scheduler allowing for a fast and easy rescheduling.
-   [Grouping](components/scheduler/grouping/?utm_medium=referral&utm_source=npm&utm_campaign=kno-ui-react-trial-npm-scheduler)&mdash;The Scheduler enables grouping by different resources like meeting rooms, teams, and more, which provides a nice arrangement of the events within the same group.
-   [Resources](components/scheduler/resources/?utm_medium=referral&utm_source=npm&utm_campaign=kno-ui-react-trial-npm-scheduler)&mdash;Define different custom resources such as available conference rooms or guest lists for the Scheduler to work with.
-   [Recurring events](components/scheduler/recurring/?utm_medium=referral&utm_source=npm&utm_campaign=kno-ui-react-trial-npm-scheduler)&mdash;You can schedule a recurring event that will occur on a specific day, for example, each Monday during the next year.
-   [Controlled and uncontrolled mode for the date selection](components/scheduler/date-selection/?utm_medium=referral&utm_source=npm&utm_campaign=kno-ui-react-trial-npm-scheduler)&mdash;This capability allows you to completely override the date selection from outside the Scheduler.
-   [Timezones](components/scheduler/timezones/?utm_medium=referral&utm_source=npm&utm_campaign=kno-ui-react-trial-npm-scheduler)&mdash;You can display the events in the Scheduler in different timezones when building multinational applications.
-   [Globalization](components/scheduler/timezones/?utm_medium=referral&utm_source=npm&utm_campaign=kno-ui-react-trial-npm-scheduler)&mdash;The globalization process combines the translation of component messages with adapting them to specific cultures
-   [Theme support](components/styling/?utm_medium=referral&utm_source=npm&utm_campaign=kno-ui-react-trial-npm-scheduler)&mdash;The knoReact Scheduler, as well as all 100+ components in the knoReact suite, are styled in three polished themes (Bootstrap, Material, and Default) and can be further customized to match your specific design guidelines.
